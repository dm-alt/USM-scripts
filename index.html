Installation Guide (Simplified)

  1. Click the "Install Script" button for the script you want.
  2. Tampermonkey will automatically open a new tab with the installation screen.
  3. Click the "Install" button on the Tampermonkey screen.
  4. Refresh Front: Reload the Front App page to activate the script.
